More options
FreeClimb Plugin Demo

Two ways the agent works with FreeClimb.

MCP

Talk to the live platform

A standard way for the agent to ask FreeClimb questions and take action: read the account, check call logs, place a call, or send a text, all using real, live data inside any AI app.

CLI

Build and run on your machine

A command-line toolkit the agent runs locally to scaffold the project, start a server, expose webhooks, and test the whole call flow end to end.

More options, one agent. MCP gives it live awareness and actions; the CLI lets it build and run real apps, so it goes from idea to a working call without anyone stitching tools together.
