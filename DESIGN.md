# Design

## Source

This demo deck uses FreeClimb's brand register: deep teal and dark ground for structure, electric green and orange as campaign accents, Open Sans typography, and flat bordered panels for tool groupings.

## Color

- Deep teal: `#00333f`
- Dark ground: `#11242d`
- Brand blue: `#004e63`
- Action yellow: `#ddb907`
- Electric green: `#c2ff18`
- Brand orange: `#fa660a`
- Mist blue: `#a1d0d3`

Use electric green and orange as brand/campaign accents. Use teal and dark ground for structure.

## Typography

Use Open Sans for the deck. Large headings can be bold and direct. Keep body copy short and readable. Avoid a technical monospace look unless showing code or command examples.

## Composition

Slides should feel more like a marketing/demo narrative than an app screen. Prefer one dominant idea per slide, large type, confident spacing, and simple visuals. The FreeClimb logo should be prominent but never duplicated on the same slide.

## Components

- Brand pills for short capability tags.
- Flat bordered panels for tool groupings.
- Large nested visuals when explaining plugin structure.
- Short numbered lists only when the content is actually sequential.

## Motion

Use subtle slide transitions only. Respect reduced-motion preferences.

## Anti-patterns

Avoid generic SaaS card grids, glassmorphism, gradient text, terminal-native developer styling, dense architecture diagrams, and repeated tiny eyebrow labels as the main page structure.
