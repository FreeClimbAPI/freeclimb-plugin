# /freeclimb-account

Manage the FreeClimb account connected to this plugin.

1. Load the `manage-freeclimb-account` skill.
2. Ask the user whether to show the connected account, connect or switch accounts, or log out.
3. Follow the selected workflow without requesting, displaying, or writing credentials in chat.
4. Verify the final state with the FreeClimb `get_account` MCP tool when connected.
